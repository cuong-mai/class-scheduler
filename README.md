<blockquote>
<ul>
 	<li>
<h6>Visualizes all possible combinations of classes (without time conflicts) to help students select the fittest schedule (from text-based information on online course enrollment systems).</h6>
</li>
 	<li>
<h6>Provides a unique feature that colleges’ course enrollment systems have been missing.</h6>
</li>
 	<li>
<h6>Visited by 94 users with 128 sessions during enrollment period of Fall 2017 (Aug - Sept).</h6>
<h6></h6>
</li>
</ul>
</blockquote>
